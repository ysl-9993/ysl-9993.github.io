## 📝 题目描述
- 假设你正在爬楼梯。需要`n`阶你才能到达楼顶。

- 每次你可以爬`1`或`2`个台阶。你有多少种不同的方法可以爬到楼顶呢？

## 🤔 解题思路
- 一道最简单的动态规划问题。
- 状态转移方程：$f(n) = f(n-1) + f(n-2)$

## 🧑‍💻 题解
```python
class Solution:
    def climbStairs(self, n: int) -> int:
        p = 0
        q = 0
        r = 1
        for i in range(n):
            p = q
            q = r
            r = p + q
        return r
```
