## 步骤
### 1. 配置博客容器统计接口
- 在`docker-compose.yml`中配置数据库连接
    ```yml
    postgres:
        image: postgres:15-alpine
        container_name: blog_pg
        volumes:
        - ./pg_data:/var/lib/postgresql/data
        environment:
        POSTGRES_USER: xxx # 数据库用户名
        POSTGRES_PASSWORD: xxx # 数据库密码
        POSTGRES_DB: visit_stats
        ports:
        - "127.0.0.1:5432:5432"
        restart: always
    ```

- 在`nginx.conf`中配置统计接口
    ```nginx
    # 统计接口（转发到宿主机统计服务）
    location /track {
        proxy_pass http://172.17.0.1:5001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # 统计后台
    location /stats {
        proxy_pass http://172.17.0.1:5001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
    ```

### 2. 配置博客统计服务
- 在博客根目录下的`index.html`中配置统计服务
    ```html
    <script>
    fetch("/track")
    </script>
    ```



### 3. 新建脚本文件
- 新建目录`/home/ubuntu/visit-stats/`
- 在目录下新建文件`track_server.py`
    ```python
    from flask import Flask, request, jsonify
    import psycopg2
    from psycopg2 import pool
    import user_agents

    app = Flask(__name__)

    DB_CONFIG = {
        "host": "127.0.0.1",
        "port": 5432,
        "user": "xxx", # 数据库用户名
        "password": "xxx", # 数据库密码
        "database": "visit_stats"
    }

    connection_pool = pool.SimpleConnectionPool(
        minconn=1, maxconn=5,** DB_CONFIG
    )

    def init_table():
        try:
            conn = connection_pool.getconn()
            c = conn.cursor()
            c.execute('''
                CREATE TABLE IF NOT EXISTS visit_log (
                    id SERIAL PRIMARY KEY,
                    ip VARCHAR(40),
                    url TEXT,
                    title TEXT,
                    referrer TEXT,
                    user_agent TEXT,
                    device VARCHAR(20),
                    create_time TIMESTAMP DEFAULT NOW()
                )
            ''')
            conn.commit()
            connection_pool.putconn(conn)
        except:
            pass

    @app.route('/track', methods=['POST'])
    def track():
        try:
            data = request.json
            ip = request.remote_addr
            url = data.get('url', '')
            title = data.get('title', '')
            referrer = data.get('referrer', '')
            ua = data.get('ua', '')

            user_agent = user_agents.parse(ua)
            device = "Mobile" if user_agent.is_mobile else "PC"

            conn = connection_pool.getconn()
            c = conn.cursor()
            c.execute('''
                INSERT INTO visit_log (ip, url, title, referrer, user_agent, device)
                VALUES (%s,%s,%s,%s,%s,%s)
            ''', (ip, url, title, referrer, ua, device))
            conn.commit()
            connection_pool.putconn(conn)
            return jsonify({"status": "ok"})
        except:
            return jsonify({"status": "error"}), 500

    @app.route('/stats', methods=['GET'])
    def stats():
        try:
            conn = connection_pool.getconn()
            c = conn.cursor()
            c.execute("SELECT COUNT(*) FROM visit_log")
            pv = c.fetchone()[0]
            c.execute("SELECT COUNT(DISTINCT ip) FROM visit_log")
            uv = c.fetchone()[0]
            c.execute("SELECT COUNT(*) FROM visit_log WHERE DATE(create_time) = CURRENT_DATE")
            today_pv = c.fetchone()[0]
            c.execute('''SELECT url, COUNT(*) as cnt FROM visit_log GROUP BY url ORDER BY cnt DESC LIMIT 10''')
            top_pages = c.fetchall()
            connection_pool.putconn(conn)
            return jsonify({"总PV": pv, "总UV": uv, "今日PV": today_pv, "页面排行": [{"url": u, "次数": c} for u, c in top_pages]})
        except:
            return jsonify({"status": "error"}), 500

    if __name__ == '__main__':
        init_table()
        app.run(host='0.0.0.0', port=5001, debug=False)
    ```

### 4. 配置服务器环境
- 创建`conda`虚拟环境（需提前安装和配置`conda`）
    ```bash
    conda create -n stats python=3.10
    ```
- 进入虚拟环境
    ```bash
    conda activate stats
    ```
- 安装依赖
    ```bash
    pip install flask psycopg2-binary user_agents
    ```

### 5. 启动统计服务
- 后台启动
    ```bash
    cd /ubuntu/visit-stats
    nohup python track_server.py > track.log 2>&1 &
    ```
    > `nohup`表示后台运行

- 检查运行状态
    ```bash
    ps aux | grep track_server
    ```

- 停止服务
    ```bash
    pkill -f track_server.py
    ```
