## 📝 题目描述
- 给定一个已排序的链表的头`head`，删除原始链表中所有重复数字的节点，只留下不同的数字 。返回 **已排序的链表** 。

## 🤔 解题思路
- 方法一：两次遍历
    - 可以先遍历一次所有节点，用哈希表存储所有节点出现的次数。
    - 然后再设置一个虚拟头节点（防止头节点被删除），再用指针`p`再次遍历链表，删除所有出现次数大于等于2的节点。
    - 最终返回虚拟头节点的下一个节点。
- 方法二：一次遍历
    - 直接在一次遍历中删除重复节点，设置一个虚拟头节点，空间复杂度可以降到`O(1)`。
    - 利用递增的性质，核心是只要找到重复的节点，就循环删除所有重复的节点。

## 🧑‍💻 题解
- 方法一：两次遍历
    - 时间复杂度：`O(n)`
    - 空间复杂度：`O(n)`
        ```python
        # Definition for singly-linked list.
        # class ListNode:
        #     def __init__(self, val=0, next=None):
        #         self.val = val
        #         self.next = next
        class Solution:
            def deleteDuplicates(self, head: Optional[ListNode]) -> Optional[ListNode]:
                # 哈希存储所有节点出现的次数
                table = {}
                p = head
                while p:
                    table[p.val] = table.get(p.val, 0) + 1
                    p = p.next
                
                dummy = ListNode()
                dummy.next = head
                p = dummy

                while p.next:
                    if table[p.next.val] >= 2:
                        p.next = p.next.next
                    else:
                        p = p.next
                
                return dummy.next
        ```
- 方法二：一次遍历
    - 时间复杂度：`O(n)`
    - 空间复杂度：`O(1)`
        ```python
        # Definition for singly-linked list.
        # class ListNode:
        #     def __init__(self, val=0, next=None):
        #         self.val = val
        #         self.next = next
        class Solution:
            def deleteDuplicates(self, head: Optional[ListNode]) -> Optional[ListNode]:
                dummy = ListNode()
                dummy.next = head
                p = dummy
                while p.next and p.next.next:
                    if p.next.val == p.next.next.val:
                        x = p.next.val
                        while p.next and p.next.val == x:
                            p.next = p.next.next
                    else:
                        p = p.next
                return dummy.next
        ```
