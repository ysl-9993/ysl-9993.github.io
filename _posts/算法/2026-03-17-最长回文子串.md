## 📝 题目描述
- 给你一个字符串`s`，找到`s`中最长的 **回文子串** 。

## 🤔 解题思路
- 如果`s[l:r+1]`是回文子串，那么`s[l+1:r-1]`也是回文子串，动态规划问题，`dp[i][j]`表示`s[i:j+1]`是否是回文子串。
- 状态转移方程：$$dp[i][j] = dp[i+1][j-1] \land (s[i] == s[j])$$
    - 将`dp[i][j]`初始化为`False`，`dp[i][i]`初始化为`True`，表示长度为1的子串都是回文子串。
    - 从长度为`2`的子串开始，找到新的回文子串，直接更新`ans`，因为后面找到的子串长度一定大于等于当前的`ans`。
    - 每个长度情况下遍历所有子串，如果`s[l] == s[r]`：
        - 若`l+1 <= r-1`，那么`dp[l][r] = dp[l+1][r-1]`；
        - 否则，`dp[l][r] = True`，因为当前`s[l:r+1]`只有两个或一个字符，所以一定是回文子串。
    - 在每个长度下遍历过程中，如果`s[l] != s[r]`，那么`dp[l][r] = False`。

## 🧑‍💻 题解
```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        n = len(s)
        ans = s[0]
        if n < 2:
            return s
        dp = [[False] * n for _ in range(n)]
        for i in range(n):
            dp[i][i] = True
        for length in range(2, n + 1):
            for l in range(0, n - length + 1):
                # length = r - l + 1
                r = length + l - 1
                if s[l] == s[r]:
                    if l + 1 <= r - 1:
                        dp[l][r] = dp[l + 1][r - 1]
                    else:
                        dp[l][r] = True
                else:
                    dp[l][r] = False
                if dp[l][r] == True:
                    ans = s[l:r+1]
        return ans
```
