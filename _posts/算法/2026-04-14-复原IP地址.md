## 📝 题目描述
- **有效 IP 地址** 正好由四个整数（每个整数位于`0`到`255`之间组成，且不能含有前导`0`），整数之间用`'.'`分隔。

- 例如：`"0.1.2.201"`和`"192.168.1.1"`是 **有效 IP 地址**，但是`"0.011.255.245"`、`"192.168.1.312"`和`"192.168@1.1"`是 **无效 IP 地址**。

- 给定一个只包含数字的字符串`s`，用以表示一个 IP 地址，返回所有可能的 **有效 IP 地址**，这些地址可以通过在`s`中插入`'.'`来形成。你 **不能** 重新排序或删除`s` 中的任何数字。你可以按 **任何** 顺序返回答案。

## 🤔 解题思路
- 可以采用递归和回溯的思路处理，整个ip地址一共有4个子段，每层递归处理一个子段，传入剩余的字符串，全局用`cache`维护当前已处理的子段。
- 在每层递归开始时判断：每当所有子段全部完成时，或字符串为空时，直接返回，若两个条件同时满足，则将当前`cache`中的子段用`'.'`连接起来  加入结果列表。
- 在每层递归开始时，如果遇到当前字符串的第一个字符为`0`，则只能独立成一个子段，直接继续递归处理剩余字符串。
- 每次递归结束后需要回溯到上一个状态，`cache.pop()`弹出最后一个子段。

## 🧑‍💻 题解
- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`
    ```python
    class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cache = []

        def dfs(k, s):
            if k == 4:
                # 所有子段全部完成
                if not s:
                    res.append(".".join(seg for seg in cache))
                return
            
            if not s:
                return

            if s[0] == "0":
                # 首个字符为0，只能独立成子段
                cache.append("0")
                dfs(k+1, s[1:])
                cache.pop()
                return
            
            cur = ""
            for char in s:
                cur += char
                if int(cur) > 255:
                    cur = cur[:-1]
                    break
                cache.append(cur)
                dfs(k+1, s[len(cur):])
                cache.pop()

        dfs(0, s)
        return res
    ```
