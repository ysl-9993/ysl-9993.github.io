## 📝 题目描述
- 给定一个不含重复数字的数组`nums`，返回其 **所有可能的全排列** 。你可以 **按任意顺序** 返回答案。


## 🤔 解题思路
- 采用回溯法，递归生成所有可能的排列。
- 递归终止条件：
    - 当`first >= n`时，说明已经生成了一个排列，将当前排列添加到结果列表中。
- 递归过程：
    - 从`first`开始遍历数组，将当前元素与`first`位置的元素交换，递归生成剩余元素的排列。
    - 递归结束后，将交换的元素恢复原位，继续尝试下一个元素。
- 易错点：
    - 每当生成一个排列时，要使用`nums[:]`来复制当前排列，而不是直接使用`nums`。

## 🧑‍💻 题解
```python
class Solution:
    def permute(self, nums: List[int]) -> List[List[int]]:
        n = len(nums)
        res = []
        def backtrace(first):
            if first >= n:
                res.append(nums[:])
            for i in range(first, n):
                nums[first], nums[i] = nums[i], nums[first]
                backtrace(first + 1)
                nums[first], nums[i] = nums[i], nums[first]
        backtrace(0)
        return res
```
