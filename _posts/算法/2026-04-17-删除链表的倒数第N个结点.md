## 📝 题目描述
- 给你一个链表，删除链表的倒数第`n`个结点，并且返回链表的头结点。

## 🤔 解题思路
- 先遍历一次链表，计算出链表长度`length`，然后将倒数位置`n`转换为正数位置`k = length - n + 1`。
- 设置虚拟头节点和指针`p`，指针`p`从虚拟头节点开始遍历链表，直到遍历到位置`k`退出循环，重新连接链表：`p.next = p.next.next`。
- 最终返回虚拟头节点的下一个节点`dummy.next`。

## 🧑‍💻 题解
- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`
    ```python
    # Definition for singly-linked list.
    # class ListNode:
    #     def __init__(self, val=0, next=None):
    #         self.val = val
    #         self.next = next
    class Solution:
        def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
            length = 0
            p = head
            while p:
                length += 1
                p = p.next

            k = length - n + 1

            cnt = 1
            dummy = ListNode()
            dummy.next = head
            p = dummy
            while cnt < k:
                p = p.next
                cnt += 1
            p.next = p.next.next

            return dummy.next
        ```
