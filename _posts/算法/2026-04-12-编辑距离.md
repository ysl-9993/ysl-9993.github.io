## 📝 题目描述
- 给你两个单词`word1`和`word2`， 请返回将`word1`转换成`word2`所使用的最少操作数。

你可以对一个单词进行如下三种操作：

- 插入一个字符
- 删除一个字符
- 替换一个字符

## 🤔 解题思路
- 因为可以对任何一个单词执行增加、删除、替换操作，等价为可以执行`word1`增加、`word2`增加、`word1`替换操作，将操作化简。
- 因此可以采用动态规划，定义状态`dp[i][j]`为`word1`前`i`个字符和`word2`前`j`个字符的编辑距离。
- 初始化：`word1`为空时，编辑距离为`j`，`word2`为空时，编辑距离为`i`。
- 定义三种情况：
    - `word1`增加一个字符：`add1 = dp[i-1][j] + 1`
    - `word2`增加一个字符：`add2 = dp[i][j-1] + 1`
    - `word1`替换一个字符：
        - 若`word1[i-1] == word2[j-1]`，则`change = dp[i-1][j-1]`
        - 若`word1[i-1] != word2[j-1]`，则`change = dp[i-1][j-1] + 1`
- 状态转移方程为：
    - 若`word1[i-1] == word2[j-1]`，则 $$dp[i][j] = min(dp[i-1][j] + 1, dp[i][j-1] + 1, dp[i-1][j-1])$$
    - 若`word1[i-1] != word2[j-1]`，则 $$dp[i][j] = min(dp[i-1][j] + 1, dp[i][j-1] + 1, dp[i-1][j-1] + 1)$$

## 🧑‍💻 题解
- 时间复杂度：`O(mn)`
- 空间复杂度：`O(mn)`
    ```python
    class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m = len(word1)
        n = len(word2)
        dp = [[0] * (n + 1) for _ in range(m + 1)]
        
        # 初始化
        for i in range(1, m + 1):
            dp[i][0] = i
        for i in range(1, n + 1):
            dp[0][i] = i
        
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                add1 = dp[i-1][j] + 1
                add2 = dp[i][j-1] + 1
                if word1[i-1] == word2[j-1]:
                    change = dp[i-1][j-1]
                else:
                    change = dp[i-1][j-1] + 1
                dp[i][j] = min(add1, add2, change)
        
        return dp[m][n]
    ```
