## 📝 题目描述
- 给你二叉树的根节点`root`，返回其节点值的 **锯齿形层序遍历** 。（即先从左往右，再从右往左进行下一层遍历，以此类推，层与层之间交替进行）。

## 🤔 解题思路
- 采用双端队列`collections.deque`，节点根据层号判断是从左往右入队还是从右往左入队，一层结束后将队列转换为列表加入结果列表。
- 需要变量维护当前层的所有节点，初始时为根节点`root`，后续每次遍历该层的所有节点并依次`pop`，结束后将下一层的所有节点入队。
- 每次遍历一层，通过`len(res)`判断当前层是奇数层还是偶数层，遍历结束后将队列转换为列表加入结果列表`res`。

## 🧑‍💻 题解
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def zigzagLevelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []
        res = []
        nodes = [root]
        while nodes:
            row = collections.deque()
            for i in range(len(nodes)):
                node = nodes.pop(0)
                if len(res) % 2 == 0:
                    # 奇数层 从左往右
                    row.append(node.val)
                else:
                    # 偶数层 从右往左
                    row.appendleft(node.val)
                if node.left:
                    nodes.append(node.left)
                if node.right:
                    nodes.append(node.right)
            res.append(list(row))
        return res
```
