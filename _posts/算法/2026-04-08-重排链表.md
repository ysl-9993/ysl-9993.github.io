## 📝 题目描述
- 给定一个单链表`L`的头节点`head`，单链表`L`表示为：

    > L0 → L1 → … → Ln - 1 → Ln
- 请将其重新排列后变为：

    > L0 → Ln → L1 → Ln - 1 → L2 → Ln - 2 → …
- 不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。


## 🤔 解题思路
- 遍历一次链表，先将所有节点存入列表中，再利用双指针从列表中重新编排链表。
- 链表重排后最后一个节点要断开，否则会形成环形链表。

## 🧑‍💻 题解
- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`
    ```python
    # Definition for singly-linked list.
    # class ListNode:
    #     def __init__(self, val=0, next=None):
    #         self.val = val
    #         self.next = next
    class Solution:
        def reorderList(self, head: Optional[ListNode]) -> None:
            """
            Do not return anything, modify head in-place instead.
            """
            table = []
            p = head
            while p:
                table.append(p)
                p = p.next
            
            n = len(table)
            l = 0
            r = n - 1
            node = ListNode()
            while l <= r:
                node.next = table[l]
                node = node.next
                if l != r:
                    node.next = table[r]
                    node = node.next
                l += 1
                r -= 1
            # 断开最后一个节点
            node.next = None
    ```
