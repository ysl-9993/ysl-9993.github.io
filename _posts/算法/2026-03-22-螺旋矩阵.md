## 📝 题目描述
- 给你一个`m`行`n`列的矩阵`matrix`请按照 **顺时针螺旋顺序** ，返回矩阵中的所有元素。

## 🤔 解题思路
- 通过变量`dire`记录当前方向，初始向右，出界或已访问及时回溯并更改方向。
- 每次找到一个节点，记录该节点并更新`cur`。
- 直到记录所有节点，即`cur == total`，代表螺旋矩阵遍历完成。

## 🧑‍💻 题解
```python
class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        rows = len(matrix)
        cols = len(matrix[0])
        flag = [[False] * cols for _ in range(rows)]
        dire = 0 # 初始向右
        total = rows * cols
        ans = []
        x = 0
        y = 0
        cur = 0 # 已记录节点
        while cur < total:
            if 0 <= x < rows and 0 <= y < cols and flag[x][y] == False:
                flag[x][y] = True
                ans.append(matrix[x][y])
                cur += 1
                if dire % 4 == 0:
                    y += 1
                elif dire % 4 == 1:
                    x += 1
                elif dire % 4 == 2:
                    y -= 1
                else:
                    x -= 1
            else:
                if dire % 4 == 0:
                    y -= 1
                elif dire % 4 == 1:
                    x -= 1
                elif dire % 4 == 2:
                    y += 1
                else:
                    x += 1
                dire += 1
                if dire % 4 == 0:
                    y += 1
                elif dire % 4 == 1:
                    x += 1
                elif dire % 4 == 2:
                    y -= 1
                else:
                    x -= 1
        return ans
```
