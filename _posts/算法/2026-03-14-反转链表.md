## 📝 题目描述
给你单链表的头节点`head`，请你反转链表，并返回反转后的链表。

## 🤔 解题思路
类似`K个一组翻转链表`，但`K个一组翻转链表`的`tail`后可能还有节点要维护，因此不能通过判断移动节点是否为`None`来终止循环，要用`prev`和`tail`是否重合判断是否翻转结束；而本题可以直接让`prev`为`None`，最终移动节点`p`为`None`停止循环。

## 🧑‍💻 题解
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        p = head
        prev = None
        while p:
            temp = p.next
            p.next = prev
            prev = p
            p = temp
        return prev
```
