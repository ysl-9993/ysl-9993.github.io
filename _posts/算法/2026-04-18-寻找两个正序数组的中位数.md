## 📝 题目描述
- 给定两个大小分别为`m`和`n`的正序（从小到大）数组`nums1`和`nums2`。请你找出并返回这两个正序数组的 **中位数** 。

- 算法的时间复杂度应该为`O(log(m+n))`。

## 🤔 解题思路
- 因为题目要求时间复杂度为`O(log(m+n))`，所以可以采用二分查找的方法解决。
- 分别拿到两个数组的长度`m`和`n`，计算出总长度`total_len = m + n`，因为两个数组内部都是正序数组，所以可以将问题转化为在长度为`total_len`的数组中找到第`k`小的元素，其中`k`对应的就是中位数的位置（注意处理奇数和偶数不同的情况）。
- 定义一个辅助函数`findKthEle(k)`，用于在两个数组中找到第`k`小的元素。用`index1`和`index2`分别表示两个数组的当前索引，每次只需要比较两个数组中第`k // 2 - 1`位置的元素，较小的那个数组可以舍弃掉前面的一部分元素，从而缩小搜索范围。
- 注意在每次循环中更新`k`的值，每次更新`index1`或`index2`之后，都要相应地减少`k`的值，以反映已经舍弃掉的元素数量。
- 在每次循环开始，都要检查`index1`和`index2`是否已经到达了数组的末尾，如果到达了末尾，就直接返回另一个数组中第`k`小的元素，在检查`index1`和`index2`都没有到达末尾时，若`k`为1时，直接返回两个数组当前索引位置的较小值，即是第`k`小的元素。

## 🧑‍💻 题解
- 时间复杂度：`O(log(m+n))`
- 空间复杂度：`O(1)`
    ```python
    class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        def findKthEle(k):
            index1, index2 = 0, 0
            while True:
                if index1 == m:
                    return nums2[index2 + k - 1]
                if index2 == n:
                    return nums1[index1 + k - 1]
                if k == 1:
                    return min(nums1[index1], nums2[index2])

                new_index1 = min(index1 + k // 2 - 1, m - 1)
                new_index2 = min(index2 + k // 2 - 1, n - 1)
                pivot1 = nums1[new_index1]
                pivot2 = nums2[new_index2]

                if pivot1 <= pivot2:
                    k -= new_index1 - index1 + 1
                    index1 = new_index1 + 1
                else:
                    k -= new_index2 - index2 + 1
                    index2 = new_index2 + 1

        m = len(nums1)
        n = len(nums2)
        total_len = m + n
        if total_len % 2 == 1:
            return findKthEle(total_len // 2 + 1)
        else:
            return (findKthEle(total_len // 2) + findKthEle(total_len // 2 + 1)) / 2
    ```
