## 📝 题目描述
- 给定一个只包括`'('`，`')'`，`'{'`，`}`，`'}'`，`'['`，`']'`的字符串`s`，判断字符串是否有效。

- 有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

## 🤔 解题思路
- 将三种类型括号以`dict`形式存储在字典中。
- 遍历字符串`s`，利用栈的性质，将左括号压栈，右括号出栈，判断是否匹配。
- 遍历过程中，若遇到右括号且栈为空，或栈顶元素与当前右括号不匹配，都直接返回`False`。
- 遍历结束后，若栈为空，则返回`True`，否则返回`False`（证明有左括号未匹配）。

## 🧑‍💻 题解
- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`
    ```python
    class Solution:
        def isValid(self, s: str) -> bool:
            stack = []
            d = {'(': ')', '{': '}', '[': ']'}
            for char in s:
                if char in d:
                    stack.append(char)
                else:
                    if len(stack) == 0:
                        return False
                    if d[stack.pop()] == char:
                        continue
                    else:
                        return False
            if len(stack) == 0:
                return True
            else:
                return False
    ```
