## 📝 题目描述
- 给你单链表的头指针`head`和两个整数`left`和`right`，其中`left <= right`。请你反转从位置`left`到位置`right`的链表节点，返回 **反转后的链表** 。

## 🤔 解题思路
- 先找到要反转的子链表的前驱节点`pre`和后继节点`succ`。
- 随后切断原链表与子链表的连接，防止反转过程中出现环形链表。
- 按照反转链表的步骤反转子链表。
- 最后将反转后的子链表拼接回原链表（此时`pre.next`指向`end`，`start.next`指向`succ`）。
- 注意：若`left`为`1`，则`pre`需要引入虚拟节点`dummy`，并令`dummy.next = head`，在反转过程中`head`会改变，所以最终返回`dummy.next`即为最终链表。

## 🧑‍💻 题解
- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`
    ```python
    # Definition for singly-linked list.
    # class ListNode:
    #     def __init__(self, val=0, next=None):
    #         self.val = val
    #         self.next = next
    class Solution:
        def reverseBetween(self, head: Optional[ListNode], left: int, right: int) -> Optional[ListNode]:
            # 虚拟节点
            dummy = ListNode()
            dummy.next = head
            # 反转链的前驱节点
            pre = dummy
            for _ in range(left - 1):
                pre = pre.next
            # 反转链的头节点
            start = pre.next
            # 反转链的尾节点
            end = start
            for _ in range(right - left):
                end = end.next
            # 反转链的后继节点
            succ = end.next
            
            # 切断原链与反转链的连接
            pre.next = None
            end.next = None
            
            # 反转
            p = start
            pre2 = None
            while p:
                temp = p.next
                p.next = pre2
                pre2 = p
                p = temp
            
            # 和原链拼接
            pre.next = end
            start.next = succ
        
            return dummy.next
    ```
