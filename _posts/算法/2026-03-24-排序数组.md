## 📝 题目描述
- 给你一个整数数组`nums`，请你将该数组升序排列。

- 你必须在 **不使用任何内置函数** 的情况下解决问题，时间复杂度为`O(nlog(n))`，并且空间复杂度尽可能小。

## 🤔 解题思路
- 手撕快速排序。
- 两路快速排序：
    - 每层递归选择一个基准元素，将基准元素和数组末尾元素交换。
    - 定义指针`i = left - 1`记录小于区的最大位置，遍历数组`[left, right-1]`，每找到一个小于等于基准元素的元素，就交换到小于区的下一个位置（`i += 1` `nums[i], nums[j] = nums[j], nums[i]`）。
    - 每次遍历结束后，将基准元素和小于区的下一个位置（`i+1`）的交换。
    - 此时分别递归小于区（`left, i`）和大于区（`i+2, right`），`i+1`是基准元素的位置，不需要递归处理。
- 三路快速排序：
    - 每层递归选择一个基准元素，将基准元素和数组末尾元素交换。
    - 定义指针`lt`记录小于区的最大位置、`gt`记录大于区的最小位置，通过`lt`和`gt`两个指针划分三路。
    - 定义指针`i`用于遍历，只要`i < gt`，就继续遍历。
    - 遍历过程：
        - 如果碰到小于基准元素的元素：更新`lt`位置，交换当前元素和`lt`位置的元素，自增`i`。
        - 如果碰到大于基准元素的元素：更新`gt`位置，交换当前元素和`gt`位置的元素，不用自增`i`，因为交换后`i`位置变成了之前没有遍历到的新元素，需要继续遍历位置`i`。
        - 如果碰到等于基准元素的元素：自增`i`。
    - 遍历结束后，将基准元素和大于区的第一个位置`gt`交换（因为`gt`位置的元素本来就属于大于区，交换后不影响）。
    - 此时分别递归小于区（`left, lt`）和大于区（`gt+1, right`），`gt`位置是基准元素的位置，不需要递归处理。

## 🧑‍💻 题解（标准快排）
```python
class Solution:
    def quick_sort(self, nums, left, right):
        if left >= right:
            return 
        index = random.randint(left, right)
        nums[right], nums[index] = nums[index], nums[right]
        i = left - 1
        for j in range (left, right):
            if nums[j] <= nums[right]:
                i += 1
                nums[j], nums[i] = nums[i], nums[j]
        nums[i+1], nums[right] = nums[right], nums[i+1]
        self.quick_sort(nums, left, i)
        self.quick_sort(nums, i+2, right)
    def sortArray(self, nums: List[int]) -> List[int]:
        self.quick_sort(nums, 0, len(nums)-1)
        return nums
```

## 🧑‍💻 题解（三路快排）
```python
class Solution:
    def quick_sort(self, nums, left, right):
        if left >= right:
            return 
        index = random.randint(left, right)
        nums[right], nums[index] = nums[index], nums[right]
        lt = left - 1 # 小于区的最大位置
        gt = right # 大于区的最小位置
        i = left
        while i < gt:
            if nums[i] < nums[right]:
                lt += 1
                nums[lt], nums[i] = nums[i], nums[lt]
                i += 1
            elif nums[i] > nums[right]:
                # 不能递增i 因为i位置换过来了一个没有遍历到的新数字
                gt -= 1
                nums[gt], nums[i] = nums[i], nums[gt]
            else:
                i += 1
        nums[gt], nums[right] = nums[right], nums[gt]
        self.quick_sort(nums, left, lt)
        self.quick_sort(nums, gt+1, right)
    def sortArray(self, nums: List[int]) -> List[int]:
        self.quick_sort(nums, 0, len(nums)-1)
        return nums
```
