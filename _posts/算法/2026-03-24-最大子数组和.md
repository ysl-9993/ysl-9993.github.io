## 📝 题目描述
- 给你一个整数数组`nums`，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

- 子数组是数组中的一个连续部分。

## 🤔 解题思路
- 状态转移方程：`dp[i] = max(dp[i-1] + nums[i], nums[i])`。
- 初始状态：`dp[0] = nums[0]`。

## 🧑‍💻 题解
```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        dp = [0] * 100010 
        dp[0] = nums[0]
        res = nums[0]
        for i in range(1, len(nums)):
            dp[i] = max(dp[i-1] + nums[i], nums[i])
            res = max(res, dp[i])
        return res
```
