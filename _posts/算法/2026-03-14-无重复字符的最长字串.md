## 📝 题目描述
给定一个字符串`s`，请你找出其中不含有重复字符的 **最长子串** 的长度。


## 🤔 解题思路
- 采用双指针维护滑动窗口，保证窗口内的字符串不重复。
- 采用哈希集合建立缓存，指针移动时，判断当前字符是否在缓存中，如果不在则`add()`，右指针右移；如果在，则`remove()`，左指针右移。
- 每次需要`remove()`时，更新当前最大长度`res = max(res, r-l)`，最终`while`结束时也要更新。

## ❔ 疑问
### 是否可以采用`len(s[l:r+1]) == len(set(s[l:r+1]))`判断当前窗口是否有重复字符？
可以，但是时间复杂度为`O(n²)`，采用哈希集合的时间复杂度为`O(n)`。

## 🧑‍💻 题解
```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        res = 0
        cache = set()
        l = 0
        r = 0
        while r != len(s):
            if s[r] not in cache:
                cache.add(s[r])
                r += 1
            else:
                res = max(res, r-l)
                cache.remove(s[l])
                l += 1
        res = max(res, r-l)
        return res
```
