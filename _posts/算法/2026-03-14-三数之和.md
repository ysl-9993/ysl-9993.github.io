## 📝 题目描述
给你一个整数数组`nums`，判断是否存在三元组`[nums[i], nums[j], nums[k]]`满足`i != j、i != k`且`j != k`，同时还满足`nums[i] + nums[j] + nums[k] == 0`。请你返回所有和为`0`且不重复的三元组。

注意：答案中不可以包含重复的三元组。


## 🤔 解题思路
- 可以先对数组进行排序，再用首尾双指针寻找符合条件的三元组。
- 从左到右遍历数组，左指针为`i+1`，右指针为数组最后一个元素。
- 如果`nums[i] > 0`，或`nums[i] + nums[r] + nums[r-1] < 0`，直接跳过当前`i`，没有继续遍历的必要。
- 注意维护`i`的范围为`[0, len(nums)-3]`，因为需要预留出`i`和`l`和`r`三个位置。
- 注意避免重复，如果`i`, `l`, `r`移动后和上一个元素相同，则继续移动。

## 🧑‍💻 题解
```python
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        nums = sorted(nums)
        res = []
        for i in range(len(nums)-2):
            if nums[i] > 0:
                # 最小的数已经大于0 跳过
                continue
            if i != 0 and nums[i] == nums[i-1]:
                # 连续两个num一样 避免重复 跳过
                continue
            l = i + 1
            r = len(nums) - 1
            if nums[i] + nums[r] + nums[r-1] < 0:
                # 最大的两个数之和都够 直接下一个i
                continue
            while l < r:
                if nums[i] + nums[l] + nums[r] < 0:
                    l += 1
                elif nums[i] + nums[l] + nums[r] > 0:
                    r -= 1
                else:
                    res.append([nums[i], nums[l], nums[r]])
                    while l < r and nums[l+1] == nums[l]:
                        l += 1
                    while l < r and nums[r-1] == nums[r]:
                        r -= 1
                    l += 1
                    r -= 1
        return res
```
