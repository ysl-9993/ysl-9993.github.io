## 📝 题目描述
- 给定一个二叉树的 **根节点** `root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

## 🤔 解题思路
- 依照题意，每一层最右侧的节点就是可以被看到的节点，可以用元组栈维护每一层的所有节点和该节点所在层的深度，用DFS遍历每一层，找到每一层最右侧的节点。
- 用一个哈希字典维护每一层最右侧的节点，键为深度，值为节点值，每次遍历到一个元组节点时，如果该深度在哈希字典中不存在，就将该节点值赋值给该深度的键，表示找到了该深度的最右侧节点。
- 每次循环结束，要检查当前节点是否有左右子节点，如果有，就将左右子节点和深度加1压入栈中（先压左子节点，再压右子节点，因为要保证下次循环时右侧节点先出栈）。

## 🧑‍💻 题解
- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`
    ```python
    # Definition for a binary tree node.
    # class TreeNode:
    #     def __init__(self, val=0, left=None, right=None):
    #         self.val = val
    #         self.left = left
    #         self.right = right
    class Solution:
        def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
            if not root:
                return []
            stack = [(root, 0)]
            depth_cache = {}
            res = []
            while stack:
                node, depth = stack.pop()
                if depth not in depth_cache:
                    depth_cache[depth] = node.val
                if node.left:
                    stack.append((node.left, depth+1))
                if node.right:
                    stack.append((node.right, depth+1))
            
            res = [val for val in depth_cache.values()]

            return res
    ```
