## 📝 题目描述
- 给你两个单链表的头节点`headA`和`headB` ，请你找出并返回两个单链表相交的起始节点。如果两个链表不存在相交节点，返回`null`。

- 题目数据 **保证** 整个链式结构中不存在环。

- 注意，函数返回结果后，链表必须 **保持其原始结构** 。

## 🤔 解题思路
- 方法一（哈希表）：
    - 先遍历其中一个链表，将所有节点存储到哈希表中。
    - 遍历另一个链表，如果当前节点在哈希表中，则说明当前节点为相交节点。
- 方法二（长度对齐法）：
    - 先遍历两个链表，分别记录两个链表的长度。
    - 计算出两个链表的长度差，将较长链表的指针移动到与较短链表相同长度的位置。
    - 同时遍历两个链表，如果当前节点相同，则说明当前节点为相交节点。
- 方法三（双指针法）：
    - 初始化两个指针`p`和`q`，分别指向两个链表的头节点。
    - 同时遍历两个链表，如果当前节点相同，则说明当前节点为相交节点，如果不同，两个指针继续移动。
    - 如果指针移动到末尾，则将指针指向另一个链表的头节点，继续遍历。
    - 最终两个指针会同时到达相交节点，若没有相交节点，两个指针会同时指向`NULL`，退出循环。

## 🧑‍💻 题解(哈希表)
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        table = set()
        p = headA
        while p:
            table.add(p)
            p = p.next
        q = headB
        while q:
            if q in table:
                return q
            else:
                q = q.next
        return None
```

## 🧑‍💻 题解(长度对齐法)
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        p = headA
        q = headB
        lenA = 0
        lenB = 0
        while p:
            lenA += 1
            p = p.next
        while q:
            lenB += 1
            q = q.next
        p = headA
        q = headB
        if lenA > lenB:
            for i in range(lenA - lenB):
                p = p.next
            q = headB
        elif lenB > lenA:
            p = headA
            for i in range(lenB - lenA):
                q = q.next
        else:
            p = headA
            q = headB
        
        while p and q:
            if p == q:
                return p
            else:
                p = p.next
                q = q.next
        return None
```

## 🧑‍💻 题解(双指针法)
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        p = headA
        q = headB
        while p != q:
            p = p.next if p else headB
            q = q.next if q else headA
        return p
```
