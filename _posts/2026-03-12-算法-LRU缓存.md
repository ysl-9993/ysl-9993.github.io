## 📝 题目描述
请你设计并实现一个满足 **LRU（最近最少使用）缓存** 约束的数据结构。
实现`LRUCache`类：
- `LRUCache(int capacity)`以 **正整数** 作为容量`capacity`初始化 LRU 缓存
- `int get(int key)`如果关键字`key`存在于缓存中，则返回关键字的值，否则返回`-1`。
- `void put(int key, int value)`如果关键字`key`已经存在，则变更其数据值`value`；如果不存在，则向缓存中插入该组`key-value`。如果插入操作导致关键字数量超过`capacity`，则应该逐出最久未使用的关键字。

函数`get`和`put`必须以`O(1)`的平均时间复杂度运行。

## 🤔 难点分析 & 解题思路
### 1. 如何快速找到指定`key`的`value`，满足`get`和`put`的`O(1)`时间复杂度？
采用哈希表和双向链表的数据结构，哈希表用于快速根据`key`找到双向链表中的对应节点，节点中存储`key`、`value`键值对和`prev`、`next`指针，用于动态更新缓存。

### 2. 如何维护缓存的动态更新，实现新增、移动、删除？
通过维护双向链表，实现缓存的动态更新：
- 新增节点（`put`新节点）：如果缓存已满，则先删除链表尾节点`tail`，再在链表头部新增节点。
- 移动节点（`get`已存在节点、`put`已存在节点）：将该节点从当前位置删除，插入到链表头部。
- 删除节点（`put`新节点导致缓存已满）：删除链表尾节点`tail`。

## ❔ 疑问
### 初始化时`tail.prev`指向了`head`，其间一直没有直接更新`tail`，是不是`tail.prev`一直指向`head`呢?
不是。在第一次新增节点时，调用`addToHead()`，此时`head.next`指向`tail`，所以`self.head.next.prev = node`就是更新`tail.prev`指向了新的节点`node`，因此改变`tail`未必必须直接通过`tail`修改。

## 🧑‍💻 题解(注释版)
```python
class DLinkedNode:
    # 定位链表数据结构
    def __init__(self, key=0, value=0):
        self.key = key
        self.value = value
        self.prev = None
        self.next = None

class LRUCache:
    def __init__(self, capacity: int):
        # 初始化缓存大小
        self.capacity = capacity
        # 初始化缓存哈希表
        self.cache = dict()
        # 定义伪头尾节点（不是真实节点）
        self.head = DLinkedNode()
        self.tail = DLinkedNode()
        # 连接伪头尾节点 形成链表
        self.head.next = self.tail
        self.tail.prev = self.head
        # 记录当前缓存占用情况
        self.size = 0

    def get(self, key: int) -> int:
        if key not in self.cache:
            # 当前key没有在缓存中
            return -1
        else:
            # 从哈希表中找到节点
            node = self.cache[key]
            # 更新缓存链表 将当前节点移到链表头部
            self.moveToHead(node)
            return node.value
    
    def put(self, key: int, value: int) -> None:
        if key not in self.cache:
            # 当前key没有在缓存 新定义节点
            node = DLinkedNode(key, value)
            # 哈希表增加键值对
            self.cache[key] = node
            # 头部增加新节点
            self.addToHead(node)
            # 缓存占用+1
            self.size += 1
            if self.size > self.capacity:
                # 缓存溢出 移除尾节点
                removed = self.removeTail()
                # 缓存占用-1
                self.size -= 1
                # 从哈希表中移除
                self.cache.pop(removed.key)
        else:
            # key在缓存 更新value
            node = self.cache[key]
            node.value = value
            # 将node移动到头部
            self.moveToHead(node)
            
    def moveToHead(self, node):
        # 先删除此结点
        self.removeNode(node)
        # 再将此结点插入头部
        self.addToHead(node)
    
    def removeNode(self, node):
        # 将node的前一个节点连接到node的后一个节点
        node.prev.next = node.next
        # 将node的后一个节点连接到node的前一个节点
        node.next.prev = node.prev
    
    def addToHead(self, node):
        # node的前一个节点指向伪头head
        node.prev = self.head
        # node的后一个节点指向head的后一个节点
        node.next = self.head.next
        # head的后一个节点的前一个节点指向node
        self.head.next.prev = node
        # head的后一个节点指向node
        self.head.next = node

    def removeTail(self):
        # 拿到尾节点
        node = self.tail.prev
        # 移除该节点
        self.removeNode(node)
        # 返回node让上层能拿到key 通过key从哈希表中同步移除node
        return node
```

## 🧑‍💻 题解(纯享版)
```python
class DLinkedNode:
    def __init__(self, key=0, value=0):
        self.key = key
        self.value = value
        self.prev = None
        self.next = None

class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = dict()
        self.head = DLinkedNode()
        self.tail = DLinkedNode()
        self.head.next = self.tail
        self.tail.prev = self.head
        self.size = 0

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        else:
            node = self.cache[key]
            self.moveToHead(node)
            return node.value
    
    def put(self, key: int, value: int) -> None:
        if key not in self.cache:
            node = DLinkedNode(key, value)
            self.cache[key] = node
            self.addToHead(node)
            self.size += 1
            if self.size > self.capacity:
                removed = self.removeTail()
                self.size -= 1
                self.cache.pop(removed.key)
        else:
            node = self.cache[key]
            node.value = value
            self.moveToHead(node)
            
    def moveToHead(self, node):
        self.removeNode(node)
        self.addToHead(node)
    
    def removeNode(self, node):
        node.prev.next = node.next
        node.next.prev = node.prev
    
    def addToHead(self, node):
        node.prev = self.head
        node.next = self.head.next
        self.head.next.prev = node
        self.head.next = node

    def removeTail(self):
        node = self.tail.prev
        self.removeNode(node)
        return node
```
