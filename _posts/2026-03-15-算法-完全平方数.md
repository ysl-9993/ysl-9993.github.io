## 📝 题目描述
- 给你一个整数`n`，返回和为`n`的完全平方数的最少数量。

- **完全平方数** 是一个整数，其值等于另一个整数的平方；换句话说，其值等于一个整数自乘的积。例如，`1`、`4`、`9`和`16`都是完全平方数，而`3`和`11`不是。

## 🤔 解题思路
- 动态规划问题。
- 定义状态`f[i]`表示和为`i`的完全平方数的最少数量，边界条件为`f[0] = 0`。
- 枚举范围内的所有的`j`，取`f[i - j ** 2] + 1`的最小值，找到完全平方数最少的那个。
- 状态转移方程：$$f[i] = \min_{j=1}^{\sqrt{i}} f[i - j^2] + 1$$

## 🧑‍💻 题解
```python
class Solution:
    def numSquares(self, n: int) -> int:
        f = [inf] * (n + 5)
        f[0] = 0
        for i in range(1, n + 1):
            j = 1
            while j ** 2 <= i:
                f[i] = min(f[i - j ** 2] + 1, f[i])
                j += 1
        return f[n]
```
