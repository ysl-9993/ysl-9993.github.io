## 📝题目描述
给你链表的头节点`head` ，每`k`个节点一组进行翻转，请你返回修改后的链表。

`k`是一个正整数，它的值小于或等于链表的长度。如果节点总数不是`k`的整数倍，那么请将最后剩余的节点保持原有顺序。

你不能只是单纯的改变节点内部的值，而是需要实际进行节点交换。

## 🤔难点分析 & 解题思路
1. 如何判断剩余节点是否足够一组翻转？
`head`指针永远指向链表中当前未处理的子链表的头节点，因此只需要判断`head`到链表尾节点是否足够`k`个节点即可。新增节点`pre`，使其永远指向`head`的前一个节点，让`tail`指针从`pre`开始向尾部移动`k`步，只要移动过程中`tail`指针没有指向`None`，就说明剩余节点足够一组翻转，且一轮移动结束后，`tail`已指向当前需要翻转的子链表尾部。

2. 如何翻转一组节点？
新增节点`prev`，使其指向子链表尾部的下一个节点，即`tail.next`，定义`p`节点从`head`开始循环处理子链表的指针方向，循环将`p.next`指向`prev`，并更新`prev`为`p`，`p`移动到`p.next`，直到`p`指向子节点`tail`，循环结束，局部翻转完成，此时将`tail`作为头节点，`head`作为尾节点return即可。

3. 如何将翻转后的子链表与没有参与翻转的链表部分拼接起来？
在子链表翻转过程中，已经实现了子链表和未处理的链表部分的拼接，因此只需要将已处理的部分和此轮翻转后的子链表进行拼接即可，因此在翻转前，需要记录已处理部分的尾节点，即`pre`，其始终指向当前子链表翻转前的头节点。

## 🧑‍💻题解(注释版)
```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def reverse(self, head: ListNode, tail: ListNode):
        # 初始化prev节点 记录每个节点翻转后需要指向的位置
        prev = tail.next
        p = head
        while prev != tail:
            # 直到p处理到当前子链表的tail 即说明局部翻转完成 此时prev和tail重合
            # 记录p节点的后续节点 避免断链
            temp = p.next
            # 更新p节点指向
            p.next = prev
            # 更新prev 作为下一个p的next
            prev = p
            # 更新p 继续循环
            p = temp
        # 局部翻转已经完成，将tail作为头部，head作为尾部返回，主函数接收
        return tail, head

    def reverseKGroup(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        # 新建hair节点 其next永远指向链表头节点
        hair = ListNode()
        hair.next = head
        # pre记录翻转前的上游链表尾 初始化为head的前一个节点
        pre = hair
        while head:
            # tail初始化为pre 用于判断是否够k个一组
            tail = pre
            for i in range(k):
                tail = tail.next
                if not tail:
                    # 不够k个一组，直接return整个链表
                    return hair.next
            # 够k个一组，此时tail指向子链表尾部
            head, tail = self.reverse(head, tail)
            # 和处理过的链表拼接
            pre.next = head
            # pre更新
            pre = tail
            # 更新新一轮子链表头部
            head = tail.next
        # 返回整个链表
        return hair.next
```

## 🧑‍💻题解(纯享版)
```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def reverse(self, head: ListNode, tail: ListNode):
        prev = tail.next
        p = head
        while prev != tail:
            temp = p.next
            p.next = prev
            prev = p
            p = temp
        return tail, head

    def reverseKGroup(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        hair = ListNode()
        hair.next = head
        pre = hair
        while head:
            tail = pre
            for i in range(k):
                tail = tail.next
                if not tail:
                    return hair.next
            head, tail = self.reverse(head, tail)
            pre.next = head
            pre = tail
            head = tail.next
        return hair.next
```
